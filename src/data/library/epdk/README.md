# epdk — EPDK belgeleri

Enerji Piyasası Düzenleme Kurumu'nun yayınladığı yönetmelikler.

**Mevcut belge sayısı:** 3 (bkz. `documents.ts`) — biri (`epdk-musteri-mulga`)
`status: 'deprecated'` ve `deprecated: true`, yerine geçen belgeyi
`replacementDocumentId: 'epdk-tuketici'` ile işaret eder.
